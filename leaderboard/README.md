# leaderboard

Generated leaderboard artifacts, built from [submissions/](submissions/).

- `results.json` — machine-readable, ranked per task
- `leaderboard.md` — human-readable, one table per task
- `leaderboard_generator.py` — the aggregation script that produces both
  (`python -m leaderboard.leaderboard_generator`, or `arena leaderboard`)

Both `results.json` and `leaderboard.md` are committed and regenerated by
running `arena leaderboard` after adding a submission — `.github/workflows/
update-leaderboard.yml` fails a PR if they're out of date relative to
`submissions/`.

See [ROADMAP.md](../ROADMAP.md) for the phase plan.
