The canonical project banner/thumbnail, embedded at the top of the root
[README.md](../README.md). Stable raw URL (works for any external site or
tool that just needs a direct image link):

```
https://raw.githubusercontent.com/harshitboots/ai-data-agent-benchmark-lab/main/assets/thumbnail.png
```

GitHub also auto-generates an Open Graph preview card for the repo itself
(no file needed) at:

```
https://opengraph.githubassets.com/1/harshitboots/ai-data-agent-benchmark-lab
```

If you want that OG card to show `thumbnail.png` instead of GitHub's
auto-generated one, upload it manually via **Settings → General → Social
preview** on GitHub.com — there's no API for this, it has to be done in
the UI.
